# GPH_treemap
This directory contains the scripts used for the analyses performed in the study "Large-scale individual tree species mapping using AVIRIS-NG and LiDAR data fusion". 
 
![Graphical abstract](https://github.com/ALambiel/GPH_treemap/blob/main/images/graphical_abstract.jpg)

This repository contains the scripts, tables, and parameters utilized for generating tree species classification for Gruyère Pays d'Enhaut Regional Nature park, located in Swiss Pre Alps. These maps are outcomes from the ValPar.CH project.

output maps are available in the [Zenodo repository ](https://zenodo.org/xxx)
