## Get LiDAR data
The LiDAR data covering our study area was downloaded in tiles from [swissSURFACE3D product](https://www.swisstopo.admin.ch/fr/modele-altimetrique-swisssurface3d#swissSURFACE3D---Download)the table provided by Swisstopo.
These files were then normalised so that they could be used for the remainder of the analysis.
